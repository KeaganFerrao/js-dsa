# Data Structures Implemented in JavaScript

## List of Implemented Data Structures:
 1. Stack
 2. Queue / Priority Queue
 3. Linked List (Singly)
 4. Binary Tree
 5. Array Utils


 ## Available Methods:

 ### Stack
 1. push()
 2. pop()
 3. peek()
 4. isEmpty()

 ### Queue / Priority Queue
 1. enqueue()
 2. dequeue()
 3. top()
 4. isEmpty()

 ### Linked List (Singly)
 1. add()
 2. insertAt()
 3. removeAt()
 4. printList()
 5. reverse()
 6. reverseRecursive()
 7. detectLoop()
 8. deleteList()
 9. traverse()

 ### Binary Tree
 1. preOrderTraversal()
 2. inOrderTraversal()
 3. postOrderTraversal()
 4. levelOrderTraversal()
 5. height()
 6. insertInLevelOrder()
 7. deleteNode()

 ### Array Utils
 1. selectionSort()
 2. bubbleSort()
 3. insertionSort()
 4. quickSort()
 5. mergeSort()
